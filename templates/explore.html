    PROGETTI OPEN SOURCE

    Hazen To Do List

    Hazen To Do List é un app per dispositivi android che permette di creare liste to do list in locale.

    Hazen Video Downloader

    Hazen Video Downloader é un software open source con licenza GPL, che permette di scaricare video da Youtube utilizzando la tecnologia ffmpeg e youtube-dl.

    Hazen Cloud

    Hazen Cloud é un semplice spazio web dove puoi caricare i tuoi file personali. Basato su NextCloud, é affidabile e gratuito.

    Hazen Bot Admin

    Hazen Bot Admin é il bot telegram interno di Hazen Tech, in sviluppo, per gestire gruppi correlati alla community.
